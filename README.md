# BetterDiscordAddons

A collection of plugins for BetterDiscord, the popular Discord client mod.

This repository contains custom **BetterDiscord plugins** designed to enhance your Discord experience with additional features and quality-of-life improvements.

## 🚀 Installation

1. Make sure you have **BetterDiscord** installed.
2. Download the desired `.plugin.js` file using the **Source** link above.
3. Place the file in your BetterDiscord plugins folder:
   - **Windows**: `%appdata%\BetterDiscord\plugins`
   - **Mac**: `~/Library/Application Support/BetterDiscord/plugins`
   - **Linux**: `~/.config/BetterDiscord/plugins`
4. Reload Discord (`Ctrl + R`) or restart the application.
5. Enable the plugin in the Plugins tab.
