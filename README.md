# BetterDiscordAddons

A collection of plugins for [BetterDiscord](https://betterdiscord.app/), the popular Discord client mod.

This repository contains my custom **BetterDiscord plugins**, designed to enhance your Discord experience with additional features and quality-of-life improvements.

## 📁 Plugins

### Available Plugins

- **ActivityFilter**  
  Hide specific activities from your Discord status so other users never see them.

- **SpotifyListenAlong**  
  Unlock Listen Along for Spotify in Discord without a Spotify Premium subscription.

> More plugins will be added over time.

## 🚀 Installation

1. Make sure you have **BetterDiscord** installed.
2. Download the desired `.plugin.js` file from the plugin's folder.
3. Place the file in your BetterDiscord plugins folder:
   - **Windows**: `%appdata%\BetterDiscord\plugins`
   - **Mac**: `~/Library/Application Support/BetterDiscord/plugins`
   - **Linux**: `~/.config/BetterDiscord/plugins`
4. Reload Discord (Ctrl + R) or restart the application.
5. Enable the plugin in the Plugins tab.
