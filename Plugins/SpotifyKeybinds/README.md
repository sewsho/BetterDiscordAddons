# SpotifyKeybinds Plugin

<div align="center">

![Version](https://img.shields.io/badge/version-1.0.0-blue.svg)

Control Spotify with custom keybinds from anywhere on your PC (even in-game!).

</div>

---

## Features

- **Native Keybind Controls**: Trigger Spotify controls even while Discord is unfocused.
- **Playback Shortcuts**: Toggle pause/play, skip tracks, and return to the previous track.
- **Volume Shortcuts**: Raise, lower, mute, and unmute Spotify volume with configurable volume steps.

## Installation

1. Download the [SpotifyKeybinds.plugin.js](https://github.com/sewsho/BetterDiscordAddons/blob/main/Plugins/SpotifyKeybinds/SpotifyKeybinds.plugin.js).
2. Place the `SpotifyKeybinds.plugin.js` file into your BetterDiscord plugins folder.
3. Enable the plugin through the BetterDiscord settings.

## Usage

1. **Link Spotify**: Make sure your Spotify account is connected in Discord.
2. **Open Settings**: Navigate to the BetterDiscord settings and find the SpotifyKeybinds plugin.
3. **Set Keybinds**: Expand the Keybinds category and assign shortcuts for playback, track skipping, volume, and mute controls.
4. **Adjust Preferences**: Expand the Preferences category to enable optional toast notifications.
5. **Control Spotify**: Press your configured keybinds from anywhere on your PC.

**Note**: Spotify must be active at least once before keybind controls are available. If you just opened Spotify, start playback in Spotify first so Discord can detect the active session.
