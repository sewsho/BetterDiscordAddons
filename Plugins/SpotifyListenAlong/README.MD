# SpotifyListenAlong Plugin

<div align="center">

![Version](https://img.shields.io/badge/version-1.0.0-blue.svg)

Unlock Listen Along for Spotify in Discord without a Spotify Premium subscription.

</div>

---

## Features

- **Free Listen Along**: Spoofs premium status so Listen Along works without Spotify Premium.

## Installation

1. Download the `SpotifyListenAlong.plugin.js`.
2. Place the `SpotifyListenAlong.plugin.js` file into your BetterDiscord plugins folder.
3. Enable the plugin through the BetterDiscord settings.

## Usage

1. **Link Spotify**: Make sure your Spotify account is connected in Discord.
2. **Enable the plugin**: Enable the SpotifyListenAlong plugin in the BetterDiscord settings.
3. **Use Listen Along**: Just right-click a friend who's listening to Spotify and pick "Listen Along".
