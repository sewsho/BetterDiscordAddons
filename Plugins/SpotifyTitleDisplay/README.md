<div align="center">

![Version](https://img.shields.io/badge/version-1.1.1-blue.svg)

Replaces the artist name with the song title on Spotify statuses, with an optional setting to show both.

</div>

## Features

- **Prioritize Titles**: Always see the song title first in Spotify statuses across Discord.
- **Customizable Display**: Choose whether to include the artist name after the song title in the plugin settings.

## Installation

1. Download the [SpotifyTitleDisplay.plugin.js](https://github.com/sewsho/BetterDiscordAddons/blob/main/Plugins/SpotifyTitleDisplay/SpotifyTitleDisplay.plugin.js).
2. Place the `SpotifyTitleDisplay.plugin.js` file into your BetterDiscord plugins folder.
3. Enable the plugin through the BetterDiscord settings.

## Usage

1. **Enable Plugin**: Once installed, ensure the plugin is enabled in BetterDiscord settings.
2. **View Statuses**: Check the member list or your direct messages to see the song titles displayed for anyone listening to Spotify.
