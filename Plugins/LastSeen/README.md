Tracks and displays when your Discord friends were last seen, updated automatically as they come online and go offline.

---

## Installation
1. Download `LastSeen.plugin.js`
2. Place it in your BetterDiscord plugins folder
3. Enable it in BetterDiscord Settings → Plugins

## Features
- Shows **Last Seen** timestamp on friend list rows
- Shows **Last Seen** on user popouts and DM panels
- Displays **Online for** duration when a friend is currently online

## Notes
- Timestamps are only recorded while the plugin is active, presence changes while Discord is closed or the plugin is disabled will not be tracked
- Only tracks friends, not all Discord users
- Use **Clear All Data** in plugin settings to reset all timestamps
